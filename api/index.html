QR Code Generation API

This API allows you to generate customized QR codes with various options, including gradient backgrounds, logos, custom text, and more.

Base URL

http://127.0.0.1:5000

Endpoint

Generate QR Code

POST /generate_qrcode

Request Parameters

Parameter Type Description
data string Required. The data to encode in the QR code.
logo file Optional. The logo image file to embed in the center of the QR code.
shape string Optional. The shape of the QR code modules. Can be square (default) or circle.
gradient boolean Optional. Whether to apply a gradient background. Default is false.
frame_width integer Optional. The width of the frame around the QR code in pixels. Default is 0 (no frame).
frame_color string Optional. The color of the frame around the QR code. Default is black.
custom_text string Optional. Custom text to display below the QR code.
text_color string Optional. The color of the custom text. Default is black.
text_font string Optional. The font of the custom text. Default is Arial.
text_size integer Optional. The size of the custom text in points. Default is 20.
bold boolean Optional. Whether to display the custom text in bold. Default is false.
italic boolean Optional. Whether to display the custom text in italic. Default is false.
underline boolean Optional. Whether to underline the custom text. Default is false.

Example Request

cURL

curl -X POST http://127.0.0.1:5000/generate_qrcode \
-F 'data=https://example.com' \
-F 'logo=@path/to/logo.png' \
-F 'shape=square' \
-F 'gradient=true' \
-F 'frame_width=10' \
-F 'frame_color=red' \
-F 'custom_text=Visit Us!' \
-F 'text_color=green' \
-F 'text_font=Arial' \
-F 'text_size=25' \
-F 'bold=true' \
-F 'italic=true' \
-F 'underline=true' \
--output qrcode.png

Example Response

The response will be a PNG image file containing the generated QR code, downloaded as qrcode.png.

Response Headers

Header Value
Content-Type image/png
Content-Disposition attachment; filename="qrcode.png"

Error Responses

If any required parameters are missing or invalid, the API will return an appropriate HTTP status code and error message.

400 Bad Request

  • Missing required parameter data.
  • Invalid parameter values.

Example Error Response

{
    "error": "Missing required parameter: data"
}

Notes

  • Ensure the logo image file is accessible when making the request.
  • The text_font parameter should be a valid font available on the server.
